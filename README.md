# Brain Tumor Classification using Machine Learning 🧠

## Overview
This project is an image classification system that detects and classifies brain tumors from MRI images. It categorizes scans into:
- Glioma Tumor
- Meningioma Tumor
- Pituitary Tumor
- No Tumor

The model is trained on labeled MRI image data and integrated with a Flask web application for user interaction.

---

## Features
- Upload MRI brain images  
- Automatic tumor classification  
- Supports multiple tumor categories  
- Flask-based web interface for prediction  

---

## Tech Stack
- Python  
- Logistic Regression, SVM (for classification) 
- Flask (for web application)  
- NumPy, Pandas  
- OpenCV (image processing)  

---

## Project Workflow
1. Image preprocessing (resizing, normalization)  
2. Feature extraction  
3. Model training (Logistic Regression / SVM)  
4. Model evaluation  
5. Integration with Flask for prediction
