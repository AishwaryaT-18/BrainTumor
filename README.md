# Brain Tumor Classification using Deep Learning

## Overview
This project is a Deep Learning-based image classification system that detects and classifies brain tumors from MRI images. It categorizes brain scans into different classes such as:
- Glioma Tumor
- Meningioma Tumor
- Pituitary Tumor
- No Tumor

The system is built using Python and trained on a labeled MRI image dataset.

---

## Features
- Upload MRI brain images
- Automatic tumor classification
- Supports multiple tumor categories
- Web interface for prediction (if Flask app is used)

---

## Tech Stack
- Python
- Logistic Regressor, SVM (Models used for classification)
- Flask (for web application)
- NumPy, Pandas, OpenCV
